# MVP 版本计划

## v0.1 文档原型

- 明确真实年包公式
- 定义指标字典
- 画出核心输入/输出页面
- 访谈 5-10 个目标用户

## v0.2 本地计算器

- 本地表单输入 Offer
- 基础现金、福利、成本计算
- 多 Offer 对比
- JSON 保存/导入/导出

## v0.3 风险与折现

- 股权/期权折现
- 通胀率、折现率、兑现概率设置
- 保守/中性/乐观三种情景
- 计算解释面板

## v1.0 本地完整版本

- 本地数据持久化
- 默认指标库
- 城市社保公积金规则配置
- 隐私说明
- 用户调研入口
- 版本化数据文件

## v1.1+ 网页端准备

- 数据端 API 设计
- 匿名贡献机制
- 公司福利模板审核
- 数据可信度评分
- 分享卡片

## 2026-07-08 路线修正

### 1.0 版本边界

- 做成本地优先/用户端优先的静态 Web App。
- 不做账户系统。
- 不上传用户 Offer 和薪资数据。
- 只提供公式、默认参数、UI 和可选公开数据。
- 第三方数据优先使用 GitHub Pages 或静态版本化文件。

### 后端版本条件

后端、账户、云端保存和匿名数据分析放到 1.0 之后。前提是隐私保护、授权、脱敏、加密、删除导出、样本量保护和免责声明都设计完成。

## 2026-07-08 MVP 优先级修正

MVP 优先级调整：

1. 单个 Offer/当前工作的完整估值体验。
2. 估值结果解释页。
3. 本地保存多个估值对象。
4. 多对象对比视图。

对比功能放在单个估值闭环之后实现，避免产品一开始被设计成表格型比较器。
